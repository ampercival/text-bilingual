Formatting Tips
  • General: Make sure the English and French versions use the same paragraphs in the same order.
  • Speech mode: Paste your text in both languages. Keep the paragraph order the same in each version.
  • Presentation mode: Start each slide with a line beginning with “#” (for example: # Slide 1 - Title). Under each slide, provide the English and French text. Follow the same paragraph order in both versions.
